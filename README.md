## How to start

1. Clone the project
2. Edit `Page1.js` & `Page2.js` (and add other Pages) in `Client\src\Components\Main\Screens` path
3. Add name of your pages in Main.js (`Client\src\Components\Main`) for routing
4. Add name of your pages to Drawer List `Client\src\Components\Main\Drawer\ListItems.js` for Show in Menu

## How to run

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
